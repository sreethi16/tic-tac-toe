# tic-tac-toe
# 🎮 Tic Tac Toe - GUI (Python Tkinter)

A simple **Tic Tac Toe game with a graphical user interface (GUI)** built using **Python and Tkinter**.  
This project allows two players to play the classic Tic Tac Toe game on a 3×3 grid.

---

## 📌 Features
- Interactive **GUI using Tkinter**
- **Two player gameplay (X vs O)**
- Displays the **current player**
- Detects and shows the **winner**
- **Reset button** to restart the game
- Simple and beginner-friendly Python code

---

## 🛠️ Technologies Used
- **Python**
- **Tkinter (Python GUI Library)**

## ▶️ How to Run the Project

1. Make sure **Python is installed** on your system.

2. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe-gui.git
```

3. Navigate to the project folder:

```bash
cd tic-tac-toe-gui
```

4. Run the program:

```bash
python tic_tac_toe.py
```

---

## 🎯 How to Play
1. The game starts with **Player X**.
2. Players take turns clicking the grid.
3. The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins.
4. Click the **Reset button** to start a new game.
